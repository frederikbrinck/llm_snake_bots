üéÆ Game Overview

Welcome to the Multiplayer Snake Game API! This is a real-time, WebSocket-based API that enables developers to create clients for a competitive snake game.

Game Rules

  • Grid: 50x50 cells with wraparound borders
  • Players: 2-8 players per game
  • Objective: Be the last snake alive OR reach length 300
  • Tick Rate: 200ms (5 FPS) for smooth gameplay
  • Fruits: Number of players - 1 fruits on board
  • Movement: Up, Down, Left, Right (cannot reverse into tail)

Key Features

  • üîÑ Real-time multiplayer gameplay
  • üåê WebSocket-based communication
  • üìä JSON message protocol
  • üéØ Language-agnostic client development
  • üìà Live game statistics
  • üîç Interactive API documentation

üåê API Endpoints

WebSocket Endpoints

WS /lobby
Player Connection Endpoint
Connect as a player to join the game lobby and participate in real-time gameplay.

URL: ws://localhost:3000/lobby?player_name={name}

Parameters:

  • player_name
(optional): Your display name in the game

Purpose: Real-time game participation and move submission

WS /gui
Spectator/Control Interface
Connect as a spectator to view game state and control lobby functions.

URL: ws://localhost:3000/gui

Purpose: Game observation, lobby management, and start control

HTTP Endpoints

GET /health
Health check endpoint to verify server status

Response: "OK"

GET /stats
Current game statistics and metrics

Response: JSON object with game statistics

{ "tick": 0, "alive_snakes": 0, "total_snakes": 0, "fruits_on_board": 0, "longest_snake_length": 0, "is_running": false, "winner_id": null }

OpenAPI specification in JSON format

Response: Complete OpenAPI 3.0.3 specification

GET /docs
Interactive API documentation (this page)

Response: HTML documentation with live testing

üí¨ Message Protocol

All WebSocket communication uses JSON messages with a type field for message identification.

Client ‚Üí Server Messages

JoinLobby

Join the game lobby with your player name

{ "type": "JoinLobby", "player_name": "YourPlayerName" }
SubmitMove

Submit your next move direction for the current game tick

{ "type": "SubmitMove", "direction": "Up" }

Valid directions: "Up", "Down", "Left", "Right"

StartGame

Start the game (GUI connection only, requires minimum 2 players)

{ "type": "StartGame" }
Ping

Keep the connection alive and test connectivity

{ "type": "Ping" }

Server ‚Üí Client Messages

LobbyJoined

Confirmation that you've successfully joined the lobby

{ "type": "LobbyJoined", "player_id": "uuid-string", "player_name": "YourPlayerName" }
LobbyState

Current lobby state with all connected players

{ "type": "LobbyState", "players": [ { "id": "uuid-string", "name": "Player1", "color_index": 0, "is_ready": true } ] }
GameStarted

Game has started with initial game state

{ "type": "GameStarted", "game_state": { /* GameState object */ }, "your_snake_id": "uuid-string" }
GameUpdate

Regular game state updates every tick (200ms)

{ "type": "GameUpdate", "game_state": { /* Updated GameState object */ } }
MoveRequest

Server requests your next move with valid directions

{ "type": "MoveRequest", "valid_directions": ["Up", "Down", "Left", "Right"], "time_limit_ms": 5000 }
GameEnded

Game has ended with final results

{ "type": "GameEnded", "winner": { /* LobbyPlayer object or null */ }, "final_state": { /* Final GameState */ } }
Error

Error message from the server

{ "type": "Error", "message": "Error description" }
Pong

Response to Ping message

{ "type": "Pong" }

üíª Code Examples

JavaScript/Node.js Client

const WebSocket = require('ws');

class SnakeGameClient {
    constructor(playerName) {
        this.playerName = playerName;
        this.ws = null;
    }

    connect() {
        const url = `ws://localhost:3000/lobby?player_name=${this.playerName}`;
        this.ws = new WebSocket(url);

        this.ws.onopen = () => {
            console.log('Connected to game server');
            this.joinLobby();
        };

        this.ws.onmessage = (event) => {
            const message = JSON.parse(event.data);
            this.handleMessage(message);
        };

        this.ws.onclose = () => {
            console.log('Disconnected from server');
        };
    }

    joinLobby() {
        this.sendMessage({
            type: 'JoinLobby',
            player_name: this.playerName
        });
    }

    handleMessage(message) {
        switch (message.type) {
            case 'LobbyJoined':
                console.log(`Joined lobby with ID: ${message.player_id}`);
                break;

            case 'MoveRequest':
                // Simple AI: move randomly
                const directions = message.valid_directions;
                const randomDirection = directions[Math.floor(Math.random() * directions.length)];
                this.submitMove(randomDirection);
                break;

            case 'GameUpdate':
                console.log(`Game tick: ${message.game_state.tick}`);
                break;

            case 'GameEnded':
                console.log('Game ended!', message.winner);
                break;
        }
    }

    submitMove(direction) {
        this.sendMessage({
            type: 'SubmitMove',
            direction: direction
        });
    }

    sendMessage(message) {
        if (this.ws && this.ws.readyState === WebSocket.OPEN) {
            this.ws.send(JSON.stringify(message));
        }
    }
}

// Usage
const client = new SnakeGameClient('MyBot');
client.connect();

Python Client

import asyncio
import websockets
import json
import random

class SnakeGameClient:
    def __init__(self, player_name):
        self.player_name = player_name
        self.player_id = None

    async def connect(self):
        uri = f"ws://localhost:3000/lobby?player_name={self.player_name}"

        async with websockets.connect(uri) as websocket:
            # Join lobby
            await self.send_message(websocket, {
                "type": "JoinLobby",
                "player_name": self.player_name
            })

            # Main game loop
            async for message in websocket:
                data = json.loads(message)
                await self.handle_message(websocket, data)

    async def handle_message(self, websocket, message):
        msg_type = message['type']

        if msg_type == 'LobbyJoined':
            self.player_id = message['player_id']
            print(f"Joined lobby with ID: {self.player_id}")

        elif msg_type == 'MoveRequest':
            # Simple AI: choose random valid direction
            valid_directions = message['valid_directions']
            direction = random.choice(valid_directions)

            await self.send_message(websocket, {
                "type": "SubmitMove",
                "direction": direction
            })

        elif msg_type == 'GameUpdate':
            game_state = message['game_state']
            print(f"Game tick: {game_state['tick']}")

        elif msg_type == 'GameEnded':
            winner = message.get('winner')
            if winner:
                print(f"Game ended! Winner: {winner['name']}")
            else:
                print("Game ended in a draw!")
            break

    async def send_message(self, websocket, message):
        await websocket.send(json.dumps(message))

# Usage
async def main():
    client = SnakeGameClient("PythonBot")
    await client.connect()

if __name__ == "__main__":
    asyncio.run(main())

Browser JavaScript

class BrowserSnakeClient {
    constructor(playerName) {
        this.playerName = playerName;
        this.ws = null;
    }

    connect() {
        const wsUrl = `ws://${window.location.host}/lobby?player_name=${this.playerName}`;
        this.ws = new WebSocket(wsUrl);

        this.ws.onopen = () => {
            console.log('Connected to game');
            this.joinLobby();
        };

        this.ws.onmessage = (event) => {
            const message = JSON.parse(event.data);
            this.handleMessage(message);
        };

        // Add keyboard controls
        document.addEventListener('keydown', (event) => {
            this.handleKeyPress(event);
        });
    }

    handleKeyPress(event) {
        const keyMap = {
            'ArrowUp': 'Up',
            'ArrowDown': 'Down',
            'ArrowLeft': 'Left',
            'ArrowRight': 'Right',
            'w': 'Up',
            's': 'Down',
            'a': 'Left',
            'd': 'Right'
        };

        const direction = keyMap[event.key];
        if (direction) {
            this.submitMove(direction);
            event.preventDefault();
        }
    }

    joinLobby() {
        this.sendMessage({
            type: 'JoinLobby',
            player_name: this.playerName
        });
    }

    handleMessage(message) {
        switch (message.type) {
            case 'LobbyJoined':
                this.updateStatus(`Joined as ${message.player_name}`);
                break;

            case 'GameUpdate':
                this.renderGame(message.game_state);
                break;

            case 'MoveRequest':
                this.updateStatus('Your turn! Use arrow keys or WASD');
                break;
        }
    }

    submitMove(direction) {
        this.sendMessage({
            type: 'SubmitMove',
            direction: direction
        });
    }

    sendMessage(message) {
        if (this.ws && this.ws.readyState === WebSocket.OPEN) {
            this.ws.send(JSON.stringify(message));
        }
    }

    updateStatus(message) {
        const statusEl = document.getElementById('status');
        if (statusEl) {
            statusEl.textContent = message;
        }
    }

    renderGame(gameState) {
        // Implement your game rendering logic here
        console.log('Game state:', gameState);
    }
}

// Usage
const client = new BrowserSnakeClient('WebPlayer');
client.connect();

üöÄ Local Development Guide

Prerequisites

  • Rust: Install from rustup.rs (requires Rust 1.70+)
  • Git: For cloning the repository
  • Web Browser: Modern browser with WebSocket support

Quick Start

# Clone the repository
git clone 
cd snake-game-server

# Build and run the server
cargo run --release

# Server will start on http://localhost:3000

Development Commands

# Run in development mode with hot reload
cargo watch -x run

# Run tests
cargo test

# Run with debug logging
RUST_LOG=debug cargo run

# Build for production
cargo build --release

Environment Variables

  • RUST_LOG: Set logging level (debug, info, warn, error)
  • SERVER_PORT: Override default port (default: 3000)
  • MAX_PLAYERS: Override maximum players per game (default: 8)
  • CORS_ORIGIN: Set allowed CORS origins (default: any)

Troubleshooting

  • Port in use: Set SERVER_PORT=3001 to use a different port
  • Build errors: Ensure you have Rust 1.70+ with rustc --version
  • WebSocket connection fails: Check firewall settings and CORS configuration
  • Performance issues: Use cargo run --release for optimized builds
